## MT7601U芯片系列Linux驱动

**注意**: *Linux 4.2版本的内核已经包含了该驱动([驱动源码](https://github.com/torvalds/linux/tree/master/drivers/net/wireless/mediatek/mt7601u)), 如果你使用>=4.2版本的内核, 请无视!*

#### 安装方法
---
```
  git clone https://github.com/raintean/mt7601.git
  ./install.sh
```
